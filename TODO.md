# Linked Lists Patterns:
Level 1 (foundations)
  ├── Dummy head node technique
  ├── Reverse a linked list (iterative)
  └── Two pointers (slow/fast)

Level 2 (builds on Level 1)
  ├── Find middle node
  ├── Remove N-th node from end
  ├── Detect cycle
  ├── Merge two sorted lists
  └── Remove duplicates

Level 3 (combines Level 1 + 2)
  ├── Reverse between L and R (LC 92)
  ├── Swap nodes in pairs (K-Group where K=2)
  ├── Palindrome check
  ├── Sort a linked list
  └── Partition / reorder

Level 4 (advanced combos)
  ├── Reverse nodes in K-Group  ← you're here
  ├── Flatten multilevel list
  ├── Deep copy with random pointers
  └── LRU Cache
