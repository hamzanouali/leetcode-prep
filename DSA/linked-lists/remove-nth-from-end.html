Remove N-th Node from End
LC #19 | Two-Pointer Gap Technique
🔴 The Problem
Given the head and an integer n, remove the n-th node from the end and return the head.
Example: [1] → [2] → [3] → [4] → [5], n = 2 → remove node 4

🧠 Strategy
1. Use a dummy node before head (handles removing head itself).
2. Move fast pointer n+1 steps ahead.
3. Move slow and fast together until fast = null.
4. slow is now right before the target → skip it!
Why n+1 gap?
We need slow to land one before the target so we can do slow.next = slow.next.next. Starting slow at dummy and advancing fast n+1 times creates exactly this gap.

💻 Code (JavaScript)
function removeNthFromEnd(head, n) { const dummy = { next: head }; let slow = dummy; let fast = dummy; // Move fast n+1 steps ahead for (let i = 0; i <= n; i++) { fast = fast.next; } // Move both until fast hits null while (fast !== null) { slow = slow.next; fast = fast.next; } // Skip the target node slow.next = slow.next.next; return dummy.next; }
🛡️ Why Dummy Node?
If n = list length, we're removing the head itself.
Without dummy, slow would need to start before head — impossible!
Dummy gives us that "before head" anchor for free.
Time: O(n)
Space: O(1)
💡 Key Insights
1. The gap technique: create a fixed gap between two pointers, then slide both. When the leader hits the end, the follower is at your target.
2. Dummy head + two pointers = two Level 1 patterns combined.
3. One pass only — no need to count the list length first.
