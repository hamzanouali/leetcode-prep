LRU Cache

LC #146 · Medium
TOC
Problem
Implement an LRU Cache class with:
constructor(capacity) — max size
get(key) — return value or -1
put(key, value) — insert/update, evict LRU if over capacity
const cache = new LRUCache(2); cache.put(1, 1); cache.put(2, 2); cache.get(1); // -> 1 cache.put(3, 3); // evicts key 2 cache.get(2); // -> -1
JavaScript
Show Hints
• Use a doubly linked list + HashMap
• Most recently used → head, least → tail
• get: move node to head, return value
• put: add/update at head, if over capacity remove tail
