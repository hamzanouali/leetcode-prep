Reverse a Linked List
LeetCode #206 — Easy | Iterative Approach
📋 Problem

🧠 Core Idea
Use 3 pointers and walk through the list once.
At each node, flip the arrow backwards.
prev ← curr next
Instead of curr → next, make curr → prev
⚙️ The 3 Operations (each iteration)
①
next = curr.next
Save next
②
curr.next = prev
Flip arrow
③
prev = curr
curr = next
Shift right
⚠️ Order Matters!
Must save next before flipping — otherwise we lose the reference to the rest of the list forever!

💻 Code (JavaScript)
function reverseList(head) { let prev = null; let curr = head; while (curr !== null) { const next = curr.next; // ① save curr.next = prev; // ② flip prev = curr; // ③ advance prev curr = next; // ③ advance curr } return prev; // new head }

🏗️ Why This Pattern Matters
This is the foundation pattern — once you master it, these all become variations:
Reverse K-Group
Same loop, but only k times
Reverse Between
Skip to m, reverse to n
Palindrome Check
Reverse 2nd half, compare
Swap Pairs
Reverse in groups of 2
💡 Key Insights
1. prev starts at null — that becomes the tail's next pointer.
2. When the loop ends, curr = null and prev = last node (new head).
3. Each node is visited exactly once — O(n) time, O(1) space. No recursion stack needed.
🎯 Interview Tip
Draw the 3 pointers on the whiteboard. Interviewers love seeing you trace through one full iteration before coding. Say: "save, flip, shift" — that's the entire algorithm.
