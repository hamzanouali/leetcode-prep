Swap Nodes in Pairs
LC #24 | K-Group Reversal where K=2
🔴 The Problem
Given a linked list, swap every two adjacent nodes.
Return the modified list. Don't modify node values — change the links.

🧠 Strategy
1. Use a dummy node before head.
2. For each pair, we have: prev → first → second → ...
3. Rewire to: prev → second → first → ...
4. Advance prev to first (which is now second in the pair).

💻 Code (JavaScript)
function swapPairs(head) { const dummy = { next: head }; let prev = dummy; while (prev.next !== null && prev.next.next !== null) { const first = prev.next; const second = first.next; // Swap first.next = second.next; second.next = first; prev.next = second; // Advance (prev moves to first, which is now 2nd) prev = first; } return dummy.next; }
⚠️ Odd Length?
If the list has an odd number of nodes, the last node has no pair → leave it alone.
The while condition prev.next.next !== null handles this — it won't enter the loop if there's only one node left.
Time: O(n)
Space: O(1)
💡 Key Insights
1. This is K-Group reversal where K=2 — the simplest case. Master this before tackling K-Group.
2. The 3-pointer swap (first.next, second.next, prev.next) is the core pattern.
3. prev advances to first (not second!) because after the swap, first is in the second position — exactly where prev needs to be for the next pair.
