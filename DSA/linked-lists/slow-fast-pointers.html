Two Pointers: Slow & Fast
a.k.a. Floyd's Tortoise & Hare | Linked Lists
🧠 Core Idea
Two pointers start at head.
Slow moves 1 step at a time. 🐢
Fast moves 2 steps at a time. 🐇
🐢
Slow
slow = slow.next
🐇
Fast
fast = fast.next.next
Fast covers ground at 2× speed → they diverge predictably
📌 App 1 — Find Middle Node
Why it works
When fast reaches the end,
slow is at the exact middle.
(fast traveled 2× distance → slow is at ½)
List: [1] → [2] → [3] → [4] → [5] → ∅

📌 App 2 — Detect Cycle
Why it works
If there's a cycle, fast can never reach ∅.
Instead, fast laps slow inside the cycle.
They must meet — like a fast runner lapping a slow one on a track.
List with cycle: 1 → 2 → 3 → 4 → 5 → back to 3

💻 Find Middle (JavaScript)
function findMiddle(head) { let slow = head; let fast = head; while (fast !== null && fast.next !== null) { slow = slow.next; // 🐢 1 step fast = fast.next.next; // 🐇 2 steps } return slow; // middle node }
💻 Detect Cycle (JavaScript)
function hasCycle(head) { let slow = head; let fast = head; while (fast !== null && fast.next !== null) { slow = slow.next; // 🐢 1 step fast = fast.next.next; // 🐇 2 steps if (slow === fast) return true; // collision! } return false; // fast hit ∅ → no cycle }
Time: O(n)
Space: O(1)
⚙️ The While Condition
We always check fast (not slow) because it's ahead:
fast !== null → handles even-length lists
(fast lands exactly on ∅)
fast.next !== null → handles odd-length lists
(fast lands on last node, can't jump 2)
Both checks needed → safe for any length!

💡 Key Insights
1. Same code, different questions — the while loop is nearly identical for all apps. Only what you do with the result changes.
2. For cycle start (LC #142): after collision, reset one pointer to head. Move both at 1 step. Where they meet = cycle entry.
3. For Nth from end: give fast an n-step head start, then move both at 1 step. When fast hits ∅, slow is at the answer.
🎯 Interview Tip
When you hear "middle", "cycle", "Nth from end", or "split list in half" — immediately think 🐢 🐇. Draw the two pointers and trace 2-3 steps. That's your signal.
