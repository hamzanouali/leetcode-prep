Remove N-th Node from End
LC #19 | Think of it as a Ruler
The Core Question
We can only walk forward through a linked list. So how do we find something that's defined by its distance from the end?
The Ruler Idea
Imagine you have a ruler that is exactly n nodes long. If you slide that ruler so its right end touches the last node, the left end points to the node to remove.
Example: list = [1,2,3,4,5], n = 2

Building the Ruler with Pointers
The Problem
We don't know where the list ends. We can't just place the ruler at the end. But we can build the ruler at the start and then slide it forward until it falls off.
Step 1: Build the Ruler
Place slow at the start. Move fast forward n steps. Now slow and fast are exactly n apart — that's your ruler!

Step 2: Slide the Ruler
Move slow and fast together, one step at a time. The gap stays fixed. When fast reaches the last node (fast.next = null), slow is sitting right before the target.

Step 3: Remove
slow.next = slow.next.next — done! Node 4 is gone.
Why This Works — In Plain English
When fast is at the last node, it has 0 nodes remaining after it. Since slow is n behind, slow has n nodes remaining after it. That means slow.next is the n-th from the end.
Edge Case: Removing the Head
If n = list length (remove the first node), slow needs to be before the head. That's impossible without a dummy node!

Add a dummy before head. Start slow there. Now the ruler technique works for every case, including removing the head.

Code (JavaScript)
function removeNthFromEnd(head, n) { const dummy = { next: head }; let slow = dummy; let fast = dummy; // Build the ruler: move fast n+1 ahead // (n+1 because slow starts at dummy, // so we need one extra step) for (let i = 0; i <= n; i++) { fast = fast.next; } // Slide the ruler until it falls off while (fast !== null) { slow = slow.next; fast = fast.next; } // slow is right before the target slow.next = slow.next.next; return dummy.next; }
Time: O(n)
Space: O(1)
The Takeaway
Can't count from the end? Build a ruler of known length, slide it to the end, and read the left edge. Two pointers with a fixed gap = a measuring stick you can slide through any linked list.
