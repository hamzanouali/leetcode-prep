Remove N-th Node from End
LC #19 | Every Single Step, Visualized
Setup
List: [1, 2, 3, 4, 5], n = 2
Goal: remove the 2nd node from the end (node 4)
We add a dummy node before head so we can handle edge cases. Both pointers start at dummy.
Phase 1: Advance fast (n+1 = 3 steps)
We move fast alone to create a gap. Why n+1? Because slow is at dummy (one spot before head), so we need one extra.

Code (JavaScript)
function removeNthFromEnd(head, n) { const dummy = { next: head }; let slow = dummy; let fast = dummy; // Phase 1: advance fast n+1 steps for (let i = 0; i <= n; i++) { fast = fast.next; } // Phase 2: move both until fast = null while (fast !== null) { slow = slow.next; fast = fast.next; } // Phase 3: skip the target slow.next = slow.next.next; return dummy.next; }
Time: O(n)
Space: O(1)
What to Remember
3 phases: (1) Create the gap by advancing fast alone, (2) Slide both to maintain the gap, (3) Delete when fast falls off. That's the whole algorithm — nothing more.
