Product of Array Except Self
LC #238 | Left × Right prefix products
The Problem
Given an array nums, return an array answer where answer[i] = product of all elements except nums[i].
Constraint: you cannot use division.
The Idea
For each position i, the answer is:
(product of everything LEFT of i) × (product of everything RIGHT of i)
Build left prefix products and right prefix products separately, then multiply them together.

How It Works
Step 1: Left Pass (left-to-right)
Start with answer[0] = 1 (nothing to the left).
For each i from 1 to n-1:
answer[i] = answer[i-1] × nums[i-1]
After left pass: answer = [1, 1, 2, 6]
Step 2: Right Pass (right-to-left)
Keep a running variable right = 1.
For each i from n-1 down to 0:
answer[i] ×= right, then right ×= nums[i]
After right pass: answer = [24, 12, 8, 6]
Code
function productExceptSelf(nums) { const n = nums.length; const answer = new Array(n); // Left pass: answer[i] = product of everything left of i answer[0] = 1; for (let i = 1; i < n; i++) { answer[i] = answer[i - 1] * nums[i - 1]; } // Right pass: multiply by product of everything right of i let right = 1; for (let i = n - 1; i >= 0; i--) { answer[i] *= right; right *= nums[i]; } return answer; }
When to Use
product without division prefix / suffix product left-right decomposition
Key Insights
1. Same idea as prefix sum
Instead of cumulative sums, we build cumulative products from both directions. The concept is identical — just swap + for ×.
2. Two passes do the trick
Left-to-right builds left products into the answer array. Right-to-left multiplies in the right products using a single running variable.
3. O(1) extra space
The output array doesn't count as extra space (per the problem). By using a running right variable instead of a separate right array, we avoid any additional allocation.
Complexity
Time: O(n)
Space: O(1) extra
