Code Template (Variable Window)
function minSubArrayLen(target, nums) { let left = 0, sum = 0, minLen = Infinity; for (let right = 0; right < nums.length; right++) { sum += nums[right]; while (sum >= target) { minLen = Math.min(minLen, right - left + 1); sum -= nums[left]; left++; } } return minLen === Infinity ? 0 : minLen; }
When to Use
"contiguous subarray" "substring" "window of size k" "minimum subarray" "maximum subarray" "with condition"
Key Insights
1. Fixed Window
Add the new right element, remove the old left element, slide one step. No need to recompute the whole window.
2. Variable Window
Expand right to satisfy the condition, then shrink left to optimize. The while loop inside the for loop is the key pattern.
3. Why O(n)?
Both pointers only move forward. Left moves at most n times total across all iterations. So inner + outer = O(n), not O(n²).
Time: O(n)
Space: O(1)
