LC #75 | Three-way partition in one pass
The Problem
Given an array of 0s, 1s, and 2s, sort it in-place using only one pass.
No extra array. No counting sort. One scan through.
Input:
2
0
2
1
1
0
The Intuition
Think of the array as having three regions that grow inward:
low = boundary of 0s (left wall), high = boundary of 2s (right wall), mid = current scanner.

mid walks left to right. At each cell:
• If 0 → swap with low, advance both
• If 2 → swap with high, shrink high (don't advance mid!)
• If 1 → just advance mid
[0..low-1] = 0s [low..mid-1] = 1s [mid..high] = ??? [high+1..end] = 2s

Code
function sortColors(nums) { let low = 0, mid = 0, high = nums.length - 1; while (mid <= high) { if (nums[mid] === 0) { [nums[low], nums[mid]] = [nums[mid], nums[low]]; low++; mid++; } else if (nums[mid] === 2) { [nums[mid], nums[high]] = [nums[high], nums[mid]]; high--; // don't advance mid! } else { mid++; // it's a 1, skip } } }
Key Insights
Why not mid++ after swap with high?
The value swapped in from high is unknown — it hasn't been examined yet. We must check it before moving on.
Why mid++ after swap with low?
Everything between low and mid has already been processed. The value swapped from low is guaranteed to be a 1 (or mid == low), so it's safe to advance.
One-pass O(n) sort for 3 values
Each element is visited at most twice (once by mid, possibly once via swap). The three pointers carve the array into sorted regions in a single scan.
When to Use

Complexity
Time: O(n) Space: O(1)
